# Nnexx Convert .ui to SGTK PyQT

This extension adds a right-click context menu to `.ui` files in the Explorer, allowing you to run a Python script to convert them into SGTK PyQT format.

## Usage
1. Right-click any `.ui` file in the Explorer.
2. Select **"Convert .ui to SGTK PyQT"**.
3. The extension will trigger your Python script (you must add the execution code in `extension.js`).

## Install
Copy the folder of this content inside the ~/.vscode/extensions/
